GWin32InputStream

GWin32InputStream — Streaming input operations for Windows file handles

Includes

#include <gio/gwin32inputstream.h>

Description

GWin32InputStream implements GInputStream for reading from a Windows file handle.

Note that &lt;gio/gwin32inputstream.h> belongs to the Windows-specific GIO interfaces, thus you have to use the gio-windows-2.0.pc pkg-config file when using it.
